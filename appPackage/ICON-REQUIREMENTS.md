# App Icon Requirements

## color.png (192×192)
- **Dimensions:** 192×192 pixels
- **Format:** PNG with full color
- **Design:** Podcast-themed icon — microphone combined with a document/page motif
- **Colors:** Use Teams accent purple (#6264A7) as primary, with white/light accents
- **Current state:** Placeholder solid-color PNG — replace with branded artwork before publishing

## outline.png (32×32)
- **Dimensions:** 32×32 pixels
- **Format:** PNG with transparent background
- **Design:** Monochrome outline version of the color icon (white lines only)
- **Colors:** White (#FFFFFF) on transparent background — Teams applies tinting automatically
- **Current state:** Placeholder transparent PNG — replace with branded outline before publishing

## Guidelines
- Follow [Teams icon design guidelines](https://learn.microsoft.com/en-us/microsoftteams/platform/concepts/build-and-test/apps-package#app-icons)
- Color icon should be visually distinct at small sizes
- Outline icon must be purely white with transparent background
